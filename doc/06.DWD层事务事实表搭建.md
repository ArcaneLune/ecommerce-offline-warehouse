# 电商离线数仓 —— DWD 层事务事实表实操文档

> 记录 DWD 层 10 张事务事实表从建表到装载的完整流程，基于 Hive on Spark 架构。
> 最后更新：2026-06-26

---

## 一、DWD 层设计概要

| 项目 | 说明 |
|------|------|
| 存储格式 | ORC + Snappy 压缩 |
| 引擎 | Hive on Spark（Iceberg 源表切 MR） |
| 数据来源 | 13 张 Iceberg 增量表 + 1 张 ODS 全量表 + 1 张日志表 |
| 分区策略 | 事务事实表按业务发生日期动态分区，全量表覆盖当日分区 |
| 建表方式 | `CREATE TABLE IF NOT EXISTS`，首次执行后不再重复 |

> ⚠️ **引擎选择**：读取 Iceberg 表（ods_*_inc）时 Spark 引擎会报 `ArrayIndexOutOfBoundsException` 或 `UnknownReason` OOM。需在 SQL 前加 `set hive.execution.engine=mr;`。Hive 原生表（ods_*_full、ods_log_inc）用 Spark 引擎正常运行。

## 三、操作步骤

### 3.1 创建 HDFS 目录

```bash
for t in dwd_trade_cart_add_inc dwd_trade_order_detail_inc dwd_trade_pay_detail_suc_inc \
         dwd_trade_cart_full dwd_trade_trade_flow_acc dwd_tool_coupon_used_inc \
         dwd_interaction_favor_add_inc dwd_traffic_page_view_inc \
         dwd_user_register_inc dwd_user_login_inc; do
    hdfs dfs -mkdir -p /warehouse/gmall/dwd/${t}
done
```

### 3.2 执行建表 DDL（首次）

```bash
hive -f /opt/software/dwd_ddl.sql
```
> 参考sql/dwd/dwd_ddl.sql

### 3.3 首日装载

```bash
hive --hivevar dt=2026-06-25 -f /opt/software/dwd_load_first.sql
```
> 参考sql/dwd/dwd_load_first.sql

### 3.4 每日装载

```bash
hive --hivevar dt=$(date +%Y-%m-%d) -f /opt/software/dwd_load_daily.sql
```
> 参考sql/dwd/dwd_load_daily.sql

## 四、验证

```bash
for t in dwd_trade_cart_add_inc dwd_trade_order_detail_inc dwd_trade_pay_detail_suc_inc \
         dwd_trade_cart_full dwd_trade_trade_flow_acc dwd_tool_coupon_used_inc \
         dwd_interaction_favor_add_inc dwd_traffic_page_view_inc \
         dwd_user_register_inc dwd_user_login_inc; do
    cnt=$(hive --silent -e "SELECT count(*) FROM ${t}" 2>/dev/null | tail -1)
    echo "${t}: ${cnt}"
done
```

## 六、与旧项目差异

| 旧项目（Maxwell CDC） | 新架构（Flink CDC 纯 INSERT） |
|----------------------|----------------------------|
| `SELECT data.id, data.user_id` | `SELECT CAST(id AS STRING), user_id` |
| `WHERE type='bootstrap-insert'` | 无 type 字段，直接读所有数据 |
| `array_contains(map_keys(old),'xxx')` | 无 old 字段，用 `WHERE xxx IS NOT NULL` 替代 |
| `skU_num = data.sku_num - old['sku_num']` | 无法计算差值，取当前值 |
