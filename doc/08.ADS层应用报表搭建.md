# 电商离线数仓 —— ADS 层应用报表实操文档

> 记录 ADS 层 16 张应用报表从建表到装载的完整流程，基于 Hive on Spark 架构。
> 最后更新：2026-06-26

---

## 一、ADS 层设计概要

| 项目 | 说明 |
|------|------|
| 存储格式 | TextFile（TAB 分隔），供 DataX 导出 MySQL |
| 引擎 | Hive on Spark（全程 Spark） |
| 数据来源 | DWS 层汇总表 + DWD 层事实表 + DIM 层维度表 |
| 分区策略 | 不分区（dt 为普通字段），INSERT OVERWRITE + UNION 自引用 |
| 建表方式 | `CREATE TABLE IF NOT EXISTS` |

> ADS 层是数仓的最上层，直接面向报表和可视化。采用 TextFile 格式便于 DataX 导出到 MySQL。数据量小，不分区，每次运行 INSERT OVERWRITE 全量覆盖。

### 核心设计

ADS 层每张表的每日装载使用 `SELECT * FROM table UNION SELECT ...` 模式：
- **首日**：用 `ads_load_first.sql`，纯 INSERT，表为空直接写入
- **每日**：用 `ads_load_daily.sql`，UNION 左侧读出旧数据，与当日新数据合并后写回

## 三、操作步骤

### 3.1 创建 HDFS 目录

```bash
for t in ads_traffic_stats_by_channel ads_page_path \
         ads_user_change ads_user_retention ads_user_stats ads_user_action \
         ads_new_order_user_stats ads_order_continuously_user_count \
         ads_repeat_purchase_by_tm ads_order_stats_by_tm ads_order_stats_by_cate \
         ads_sku_cart_num_top3_by_cate ads_sku_favor_count_top3_by_tm \
         ads_order_to_pay_interval_avg ads_order_by_province ads_coupon_stats; do
    hdfs dfs -mkdir -p /warehouse/gmall/ads/${t}
done
```

### 3.2 执行建表 DDL（首次）

```bash
hive -f /opt/software/ads_ddl.sql
hive -e "SHOW TABLES" | grep "^ads_"
# 期望输出 16 张表
```
> 参考sql/ads/ads_ddl.sql

### 3.3 首日装载

```bash
hive --hivevar dt=2026-06-26 -f /opt/software/ads_load_first.sql
```
> 参考sql/ads/ads_load_first.sql


### 3.4 每日装载

```bash
hive --hivevar dt=$(date +%Y-%m-%d) -f /opt/software/ads_load_daily.sql
```
> 参考sql/ads/ads_load_daily.sql

> 首日和每日已拆分为两个文件。每日文件通过 `SELECT * FROM ads_xxx UNION SELECT ...` 保留历史数据。

## 六、与旧项目关键差异

| 旧项目 | 新架构 | 原因 |
|--------|--------|------|
| `dws_user_user_login_td` 含 `login_date_first` | 已补齐此列 | ADS 留存率/用户统计需要 |
| `dws_trade_province_order_1d` 含 `iso_3166_2` | 已补齐此列 | ADS 省份统计可视化需要 |
| `dws_tool_user_coupon_coupon_used_1d` | 改从 DWD + DIM 直接取 | 我们无用户粒度优惠券 DWS |
| 省份 nd 聚合用 `dws_trade_province_order_nd` | ADS 内从 1d 表 inline 聚合 | 简化 DWS 层级 |
| 收藏 Top3 含 `tm_id/tm_name` | LEFT JOIN `dim_sku_full` | DWS 收藏表不含品牌维度 |
| `to_unix_timestamp()` | `unix_timestamp()` | Hive 无 to_ 前缀 |
| `nvl()` | `COALESCE()` | 兼容性 |
| 硬编码日期 | `${hivevar:dt}` | 参数化 |
| `DROP TABLE IF EXISTS` | `CREATE TABLE IF NOT EXISTS` | 不删表 |
| 单文件装载 | 首日 / 每日分离 | 与 DIM/DWD/DWS 一致 |
